Git/GitHub

Potencialize seu Desenvolvimento com Git e GitHub!

Git é um sistema de controle de versão distribuído que permite gerenciar e rastrear alterações no código de forma eficiente. GitHub, uma plataforma de hospedagem para repositórios Git, amplia essas capacidades com ferramentas poderosas para colaboração, como controle de acesso, pull requests, e automação de workflows. Juntos, Git e GitHub facilitam o desenvolvimento de software de maneira organizada e segura, potencializando a colaboração e a produtividade de equipes de todos os tamanhos.

Comandos do Git

1. init: Inicializa um repositório Git novo ou vazio.

2. clone: Clona um repositório Git existente para um novo diretório localmente.

3. branch: Lista, cria ou deleta branches.

4. checkout: Muda entre branches ou restaura arquivos de trabalho.

5. status: Mostra o estado atual do repositório Git.

6. diff: Mostra as diferenças entre versões dos arquivos.

7. add: Adiciona arquivos modificados ao índice para commit.

8. commit: Registra as mudanças no repositório.

9. push: Envia commits locais para um repositório remoto.

10. pull: Atualiza o repositório local com commits do repositório remoto.

11. revert: Desfaz um commit específico.

12. merge: Combina alterações de diferentes branches.

13. stash: Guarda temporariamente mudanças locais.

14. rm: Remove arquivos do controle de versão.

15. config: Configura opções do Git.

16. reset: Desfaz mudanças para um commit específico.

17. remote: Gerencia repositórios remotos.

18. fetch: Busca objetos e referências de um repositório remoto.

19. show: Mostra informações sobre commits, tags ou objetos.

20. help: Fornece ajuda sobre os comandos Git.

21. rebase: Aplica mudanças de um branch sobre outro.

22. blame: Mostra quem modificou cada linha de um arquivo.

23. tag: Cria, lista ou verifica tags associadas a commits.

Configuração e Acesso ao GitHub

Para começar no GitHub, siga estes passos simples:

  1. Crie uma conta no GitHub.
  2. Instale o Git no seu computador.
  3. Configure seu nome de usuário e e-mail no Git:
                git config --global user.name "Seu Nome"
                git config --global user.email "seu-email@example.com"

Para criar um repositório no GitHub:

  1. Logue no GitHub e clique em "New".
  2. Preencha as informações necessárias e crie o repositório.

Para clonar um repositório existente:

git clone https://github.com/nome-de-usuario/nome-do-repositorio.git

Após fazer alterações, use estes comandos para enviar as mudanças:

                git add .
                git commit -m "mensagem do commit"
                git push origin main

Agora você está pronto para colaborar e gerenciar projetos no GitHub!
